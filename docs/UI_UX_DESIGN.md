# LARP Protocol - Complete UI/UX Design System

## Design Philosophy

**Dark Mode First**: Streamer-friendly, easy on the eyes during long sessions
**Theatrical Aesthetic**: Stage lighting, spotlights, curtains, masks
**Gaming DNA**: Character select screens, skill trees, loot boxes
**Web3 Native**: Wallet integration, token displays, blockchain transparency

**Color System**: LARP Purple (#8B5CF6) + LARP Gold (#F59E0B) + Deep Black (#0F172A)

---

## Typography

**Headings**: DM Sans Bold
- H1: 48px, Bold
- H2: 36px, Bold
- H3: 28px, Bold
- H4: 20px, Bold

**Body**: Inter
- Regular: 16px
- Small: 14px
- Tiny: 12px

**Monospace**: JetBrains Mono (for addresses, tickers, code)
- Wallet addresses: 14px
- Token amounts: 16px Bold
